# Instrucoes locais para o Codex

Todo trabalho realizado pelo Codex neste repositorio depois do commit `803eb67` deve ser registrado em `CODEX_HANDOFF.md`.

Antes de encerrar uma mudanca:

1. Adicione ou atualize a entrada correspondente no registro.
2. Informe os arquivos alterados e as validacoes executadas.
3. Mantenha o estado `AGUARDANDO CLAUDE CODE` ate existir revisao real feita no Claude Code.
4. Nao declare alinhamento, aprovacao ou revisao pelo Claude Code sem evidencia registrada.
5. Preserve o marco e o historico do registro; correcoes devem ser novas entradas, nao apagamento das anteriores.

Testes aprovados nao substituem a revisao conceitual solicitada pelo proprietario do projeto.
