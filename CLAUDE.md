# Revisao obrigatoria das mudancas do Codex

Quando este repositorio voltar a ser trabalhado no Claude Code, leia `CODEX_HANDOFF.md` antes de modificar o sistema.

Todas as entradas com estado `AGUARDANDO CLAUDE CODE` precisam de revisao independente. Compare as mudancas com o commit-base indicado, confira se seguem a arquitetura e a linha de raciocinio dos ciclos anteriores, execute os testes relevantes e procure regressao conceitual mesmo quando a suite estiver verde.

Para concluir uma revisao:

1. Registre em `CODEX_HANDOFF.md` a data, o commit ou diff analisado e a conclusao.
2. Liste problemas encontrados e correcoes exigidas.
3. Altere uma entrada para `REVISADO PELO CLAUDE CODE` somente depois da revisao efetiva.
4. Nao apague o historico de autoria ou as limitacoes documentadas.

O objetivo e revisao cruzada, nao confirmacao automatica do trabalho anterior.
